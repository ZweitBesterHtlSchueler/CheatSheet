CSS

Name the components of a css rule properly

A css rule is build of three components. A selector, a property and a value ( selector{property: value;}).

Use combinators properly: direct child, descendant, adjacent sibling, sibling

direct child: It selects all elements that are the children of a specified element.(Combinator = >)
descendant selector: It matches all elements that are descendants of a specified element.(Combinator = space)
adjacent sibling: It selects all elements that are the adjacent siblings of a specified element. Sibling elements must have the same parent element, and "adjacent" means "immediately following".(Combinator = +)
General sibling: It selects all elements that are siblings of a specified element.(Combinator = ~)
Syntax: 

    specified element  Combinator  element
    {
      code
    }

Specify colors by color names and in hexadecimal notation

A hexadecimal color is specified with: #RRGGBB, where the RR (red), GG (green) and BB (blue) hexadecimal integers specify the components of the color.
All values must be between 00 and FF.
A color name is specified with the name of the color(red,blue,powderblue,tomato etc.).

Declarations for color, background color

You can declare a color or background-color inside the head element with the style element or inside the start tag of a element like h1-h6.
The syntax inside of the tag is < p style:"background-color/color:color;" >.

Pseudo classes for the < a > element

a:link = unvisited link(must be the first initialised)
a:visited = visited link
a:hover = mouse over link
a:active = selected link

Declaration for background image

To declare a background image you habe to write "background-image:url("imageName.type");"

Declaration for text alignment (left, right, center, justify)

You can declare a text alignment inside of the style tag or you can declare it inside of an start tag with the style attribute.
When you want to declare it inside of the style tag, you can declare it in a class or inside of elements with the Syntax: "text-align: value".
When you want to declare it inside of an start tag you can declare it with the attribute style="text-alignment:value".

Declaration for text decoration (overline, underline, line-through)

You can declare a text decoration inside of the style tag or you can declare it inside of an start tag with the style attribute.
When you want to declare it inside of the style tag, you can declare it in a class or inside of elements with the Syntax: "text-decoration: value".
When you want to declare it inside of an start tag you can declare it with the attribute style="text-decoration:value".

Declaration for text transformation (uppercase, lowercase, capitalize)

You can declare a text transformation inside of the style tag or you can declare it inside of an start tag with the style attribute.
When you want to declare it inside of the style tag, you can declare it in a class or inside of elements with the Syntax: "text-transformation: value".
When you want to declare it inside of an start tag you can declare it with the attribute style= "text-transformationt:value".

You can declare font-families inside of the style tag or you can declare it inside of an start tag with the style attribute.
